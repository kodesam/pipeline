To create an Nginx pod in Kubernetes, you can use a YAML manifest like the following example:

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: nginx-pod
spec:
  containers:
  - name: nginx
    image: nginx
    ports:
    - containerPort: 80
```

Save this YAML to a file, for example, `nginx-pod.yaml`, and apply it using the `kubectl apply` command:

```bash
kubectl apply -f nginx-pod.yaml
```

This will create an Nginx pod with a single container running Nginx using the official Nginx image. The pod listens on port 80 for incoming traffic.
