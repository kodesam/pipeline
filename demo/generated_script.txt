To create an nginx pod in Kubernetes, you can use a YAML manifest file like the following:

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: nginx-pod
spec:
  containers:
    - name: nginx
      image: nginx:latest
```

Save the above content in a file named `nginx-pod.yaml` and apply it using the `kubectl apply` command:

```bash
kubectl apply -f nginx-pod.yaml
```

This will create a pod named `nginx-pod` running the latest version of the nginx image.
